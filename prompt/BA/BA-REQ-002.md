# User Stories

## Objective
Generate User Stories based on the feature area and target users described in the user input below.

## Format
> As a **[user type]**, I want **[feature/goal]**, so that **[benefit/value]**.

## For Each Story Include

- **Acceptance Criteria** — bullet list of conditions that must be true
- **Priority** — High / Medium / Low
- **Story Points** — estimated effort (1, 2, 3, 5, 8)
- **Dependencies** — other stories or systems this depends on

## Instructions
- Cover both happy path and edge cases
- Group stories by feature or epic
- Keep each story independently deliverable where possible
