# Project Charter

## Objective
Generate a comprehensive Project Charter document for the project described in the user input below.

## Required Sections

1. **Project Overview**
   - Project name, sponsor, and brief description

2. **Business Objectives**
   - Why this project exists and what business problem it solves

3. **Project Scope**
   - In-Scope: what will be delivered
   - Out-of-Scope: what is explicitly excluded

4. **Key Stakeholders**
   - Name, role, and responsibility for each stakeholder

5. **High-Level Timeline**
   - Major phases and estimated dates

6. **Success Criteria**
   - Measurable outcomes that define project success

7. **Risks and Assumptions**
   - Known risks with likelihood/impact and mitigation
   - Assumptions the project is based on

## Instructions
- Keep language clear and executive-friendly
- Be specific about scope boundaries to avoid scope creep
- Success criteria must be measurable
