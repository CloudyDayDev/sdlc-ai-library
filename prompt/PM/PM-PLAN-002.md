# Project Timeline

## Objective
Generate a detailed Project Timeline document for the project described in the user input below.

## Required Sections

1. **Project Phases**
   - Name, goal, and duration of each phase

2. **Major Milestones**
   - Milestone name, target date, and deliverable

3. **Deliverables per Phase**
   - List of concrete outputs for each phase

4. **Dependencies**
   - Which tasks/milestones depend on others
   - External dependencies (third parties, approvals)

5. **Resource Allocation Overview**
   - Team roles involved per phase
   - Estimated effort per role

## Instructions
- Present as a table or structured list where possible
- Flag any critical path items
- Note buffer time between phases
